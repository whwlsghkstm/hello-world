소스트리

스테이지(add)
- 커밋하고자하는 파일을 스테이지에 올려(add) 커밋한다.

커밋(commit)
- 커밋 후에 언제든지 커밋한 시점으로 되돌아 갈 수 있음.
- 커밋하고싶은 파일만 묶어서 커밋실행

github에 업로드(push)

clone : 원격 저장소 복사
add : 스테이지 영역에 작업 파일 추가
commit : 세이브, 스테이지 영역의 파일들을 가지고 커밋(=save)을 만들 수 있다.
push : 원격 저장소에 커밋을 업로드한다.
